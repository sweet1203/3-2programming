## 🌟 도입 - 우리 반 시간표 떠올리기

여러분의 시간표는 어떻게 생겼나요? 📅

|       | 월   | 화   | 수   | 목   | 금   |
|-------|------|------|------|------|------|
| 1교시 | 국어 | 수학 | 영어 | 정보 | 체육 |
| 2교시 | 수학 | 영어 | 정보 | 국어 | 음악 |
| 3교시 | 영어 | 정보 | 국어 | 수학 | 미술 |

이 시간표는 **가로(요일)** 와 **세로(교시)** 두 방향으로 정보가 있죠?  
이렇게 **2개의 방향**을 가진 데이터를 **2차원 데이터**라고 해요!

> 💡 **차원이란?**  
> - **1차원**: 한 줄 (예: 학생 이름 명단)  
> - **2차원**: 표, 격자 (예: 시간표, 출석부)  
> - **3차원**: 입체 (예: 큐브, 동영상)

## 📚 핵심 개념 1: 1차원 vs 2차원

### 🔹 1차원 리스트 (한 줄)

```python
# 학생 5명의 키 (한 줄로 나열)
heights = [160, 165, 158, 170, 162]
```

이런 식으로 한 줄에 쭉~ 나열된 데이터예요.

### 🔹 2차원 리스트 (표 형태)

```python
# 학생 3명의 정보: [키, 몸무게, 사이즈]
students = [
    [160, 50, 'S'],   # 1번 학생
    [165, 55, 'M'],   # 2번 학생
    [170, 60, 'L']    # 3번 학생
]
```

**리스트 안에 리스트가 들어있죠?** 이게 바로 2차원 리스트예요!

## 📚 핵심 개념 2: 2차원 리스트 접근하기

### 🔹 인덱스 두 개 사용하기

```python
class1 = [[95, 250, "M"], [85, 230, "S"]]
#         └─ 0번 학생 ─┘  └─ 1번 학생 ─┘
#         [0][0] [0][1] [0][2]
```

```python
# 1번 학생(인덱스 0)의 상의 치수(인덱스 0)
print(class1[0][0])   # 95

# 2번 학생(인덱스 1)의 신발 치수(인덱스 1)
print(class1[1][1])   # 230

# 2번 학생(인덱스 1)의 하의 사이즈(인덱스 2)
print(class1[1][2])   # 'S'
```

### 🔹 값 수정하기

```python
class1 = [[95, 250, "M"], [85, 230, "S"]]

# 2번 학생의 상의 치수를 100으로 변경
class1[1][0] = 100
print(class1)
# 출력: [[95, 250, 'M'], [100, 230, 'S']]
```

> 📌 **기억하세요!** `리스트명[행][열]` 순서로 접근해요.

## 📚 핵심 개념 3: 2차원 리스트 값 수정하기

### 🔹 특정 칸의 값 바꾸기

```python
# 영화관 좌석표 (0=빈자리, 1=예매됨)
seats = [
    [0, 0, 0],   # A열
    [0, 0, 0],   # B열
]

# B열(인덱스 1) 2번(인덱스 1) 좌석 예매
seats[1][1] = 1
print(seats[1])   # [0, 1, 0]
```

> 💡 **수정도 접근과 같아요!**  
> `리스트명[행][열] = 새값` 으로 원하는 칸만 바꿀 수 있어요.

## 💻 핵심 코드 1: 2차원 리스트 만들기와 사용

```python
# 우리 반 시간표 (3개 요일 × 4교시)
timetable = [
    ['국어', '수학', '영어', '정보'],   # 월요일
    ['수학', '영어', '정보', '국어'],   # 화요일
    ['영어', '정보', '국어', '수학']    # 수요일
]

# 월요일 1교시 과목
print('월요일 1교시:', timetable[0][0])    # 국어

# 화요일 3교시 과목
print('화요일 3교시:', timetable[1][2])    # 정보

# 시간표 전체 출력
days = ['월', '화', '수']
for i in range(3):
    print(f'{days[i]}요일:', timetable[i])
```

## 💻 핵심 코드 2: 시간표 출력하기

```python
# 우리 반 시간표 (3개 요일 × 4교시)
timetable = [
    ['국어', '수학', '영어', '정보'],   # 월요일
    ['수학', '영어', '정보', '국어'],   # 화요일
    ['영어', '정보', '국어', '수학']    # 수요일
]

# 특정 시간 꺼내기
print('월요일 1교시:', timetable[0][0])    # 국어
print('화요일 3교시:', timetable[1][2])    # 정보

# 수요일 4교시를 '체육'으로 바꾸기
timetable[2][3] = '체육'
print('수요일:', timetable[2])
```

### 🔍 코드 풀이
| 코드 | 뜻 |
|------|-----|
| `timetable[0]` | 월요일 전체 (0번 행) |
| `timetable[0][0]` | 월요일 1교시 (0행 0열) |
| `timetable[1][2]` | 화요일 3교시 (1행 2열) |

## 🍎 실생활 사례

### 사례 1: 영화관 좌석 정보 🎬
```python
# 3행 × 4열 영화관 좌석 (0=빈자리, 1=예매됨)
seats = [
    [0, 0, 1, 1],   # A열
    [1, 0, 0, 1],   # B열
    [0, 1, 1, 0]    # C열
]

# B열 2번 좌석 상태 확인
print('B2 좌석:', '예매됨' if seats[1][1] == 1 else '빈자리')
```

### 사례 2: 학생 키 측정 기록 📏
```python
# 학생 3명 × 월별 키 (3월, 6월, 9월)
heights = [
    [160, 162, 163],   # 민지
    [165, 166, 168],   # 서연
    [158, 159, 161]    # 하늘
]

# 서연(1번)의 9월(2번) 키
print('서연 9월 키:', heights[1][2])   # 168

# 민지(0번)의 6월(1번) 키
print('민지 6월 키:', heights[0][1])   # 162
```

### 사례 3: 미니 게임 맵 🗺️
```python
# 3×3 게임 맵 (0=빈칸, 1=벽, 2=플레이어)
game_map = [
    [1, 1, 1],
    [1, 2, 0],
    [1, 0, 0]
]

# 플레이어 위치 (1행 1열)
print('플레이어 위치값:', game_map[1][1])   # 2

# 플레이어를 오른쪽(1행 2열)으로 이동
game_map[1][1] = 0   # 원래 칸 비우기
game_map[1][2] = 2   # 새 칸으로 이동
print('이동 후 1행:', game_map[1])   # [1, 0, 2]
```

## 📌 오늘 배운 내용 한눈에 정리

```
🎁 오늘의 핵심 요약

1. 차원의 의미
   ├─ 1차원: 한 줄 (예: [1, 2, 3])
   └─ 2차원: 표 형태 (예: [[1, 2], [3, 4]])
              → 리스트 안에 리스트!

2. 2차원 리스트 만들기
   data = [
       [행0_값0, 행0_값1],   # 0번 행
       [행1_값0, 행1_값1]    # 1번 행
   ]

3. 핵심 조작법
   ├─ 읽기:  data[행][열]
   └─ 수정:  data[행][열] = 새값
              (인덱스는 항상 0부터!)

4. 활용 예시
   시간표, 영화관 좌석, 시험 점수표, 게임 맵 등
```
