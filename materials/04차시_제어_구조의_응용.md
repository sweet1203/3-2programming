## 🌟 도입 - 우리 일상에서 찾는 제어 구조

여러분의 아침을 떠올려 보세요. ☀️

**아침에 일어나서…**
- **"비가 오면 우산을 챙기고, 안 오면 그냥 나간다"** → 이것이 바로 **선택 구조!** (if~else)
- **"이를 닦을 때 칫솔질을 20번 한다"** → 이것이 바로 **반복 구조!** (for)
- **"매일 아침 학교 갈 준비를 한다"** → 매일 똑같은 일을 묶어둔 **함수!** (def)

> 💡 컴퓨터도 우리처럼 **조건에 따라 판단**하고, **같은 일을 반복**하고, **자주 쓰는 기능을 묶어**서 사용해요!

## 📚 핵심 개념 1: 선택 구조 (if~elif~else)

### 🔹 기본 구조

```python
if 조건:
    조건이 참(True)일 때 실행할 코드
elif 다른조건:
    위 조건은 거짓이고 이 조건이 참일 때
else:
    위 조건들이 모두 거짓일 때
```

> ⚠️ **주의!** `if`, `elif`, `else` 다음에는 **콜론(`:`)** 이 꼭 있어야 해요!  
> 그리고 실행할 코드는 **들여쓰기(스페이스 4칸 또는 Tab)** 가 필수예요!

### 🔹 예시: 양수, 음수 판단

```python
number = int(input('숫자를 입력하세요: '))

if number > 0:
    print('양수', end='')
elif number < 0:
    print('음수', end='')
else:
    print('0', end='')
print('입니다.')
```

### 🔍 코드 풀이
- `int(input(...))` : 입력받은 문자열을 정수로 변환
- `end=''` : print 후에 줄바꿈을 하지 말라는 옵션
- 세 개 중 **하나만** 실행돼요!

## 📚 핵심 개념 2: 반복 구조 (for)

### 🔹 기본 구조

```python
for 변수 in 반복할_범위:
    반복 실행할 코드
```

### 🔹 `range()` 함수와 함께 사용

```python
# 0부터 4까지 출력 (5는 포함 안 됨!)
for i in range(5):
    print(i)
# 출력: 0, 1, 2, 3, 4

# 1부터 10까지 출력
for i in range(1, 11):
    print(i)

# 2씩 증가 (0, 2, 4, 6, 8)
for i in range(0, 10, 2):
    print(i)
```

### 🔹 리스트와 함께 사용

```python
fruits = ['사과', '바나나', '딸기']
for fruit in fruits:
    print(fruit, '먹어요!')

# 출력:
# 사과 먹어요!
# 바나나 먹어요!
# 딸기 먹어요!
```

> 📌 **꿀팁!** `range(n)`은 0부터 n-1까지예요. n은 포함되지 않아요!

## 📚 핵심 개념 3: 중첩 제어 구조

**중첩**이란 제어 구조 안에 또 다른 제어 구조가 들어있는 거예요.

### 🔹 반복문 안에 선택문

```python
# 1부터 10까지 중 짝수만 출력
for i in range(1, 11):
    if i % 2 == 0:        # 2로 나눈 나머지가 0이면 (짝수)
        print(i, '짝수')
```

> 💡 `%`(나머지 연산자): 나눗셈 후 남는 값. 짝수/홀수 판별에 자주 써요!

### 🔹 반복문 안에 반복문 (이중 반복문)

```python
# 구구단 2단~3단
for dan in range(2, 4):           # 2, 3
    for i in range(1, 10):        # 1~9
        print(f'{dan} x {i} = {dan*i}')
    print('---')
```

## 📚 핵심 개념 4: 사용자 정의 함수 (def)

### 🔹 함수가 뭐예요?

**자주 쓰는 코드를 묶어둔 것**이에요. 이름을 붙여놓고 필요할 때 불러서 사용해요.

> 비유하자면, 라면 끓이는 과정을 "라면 끓이기"라고 이름 붙여두고,  
> "라면 끓이기!"라고 외치면 자동으로 끓여주는 거예요. 🍜

### 🔹 함수 만들기 (정의)

```python
def 함수이름(매개변수):
    실행할 코드
    return 돌려줄_값      # 생략 가능
```

### 🔹 간단한 예시

```python
# 함수 정의
def say_hello(name):
    print(f'안녕하세요, {name}님!')

# 함수 사용 (호출)
say_hello('민지')      # 안녕하세요, 민지님!
say_hello('서연')      # 안녕하세요, 서연님!
```

### 🔹 값을 돌려주는 함수

```python
def add(a, b):
    result = a + b
    return result

total = add(3, 5)
print(total)      # 8
```

## 💻 핵심 코드: 종합 예시

### 사례: 구구단 출력 함수

```python
# 특정 단의 구구단을 출력하는 함수
def print_dan(dan):
    print(f'=== {dan}단 ===')
    for i in range(1, 10):            # 1부터 9까지 반복
        print(f'{dan} x {i} = {dan * i}')

# 함수 사용하기
print_dan(3)    # 3단 출력
print_dan(7)    # 7단 출력
```

```
출력 결과:
=== 3단 ===
3 x 1 = 3
3 x 2 = 6
...
3 x 9 = 27
```

> 💡 **함수의 장점!**  
> `print_dan(3)`, `print_dan(7)` 처럼 단 번호만 바꾸면 같은 코드를 재사용할 수 있어요.  
> 함수 없이 2단~9단을 모두 쓰려면 코드가 8배로 늘어나겠죠? 😅

## 📌 오늘 배운 내용 한눈에 정리

```
🎁 오늘의 핵심 요약

1. 선택 구조 (if ~ elif ~ else)
   ├─ 조건이 참이면 실행
   ├─ 콜론(:)과 들여쓰기 필수!
   └─ 비교 연산자: >, <, >=, <=, ==, !=

2. 반복 구조 (for)
   ├─ for 변수 in 범위:
   ├─ range(시작, 끝, 증가폭)
   └─ 끝 값은 포함 안 됨!

3. 중첩 제어 구조
   └─ 반복문 안에 if, 반복문 안에 반복문

4. 사용자 정의 함수 (def)
   ├─ def 함수이름(매개변수):
   ├─ return 으로 값 돌려주기
   └─ 같은 코드를 재사용할 수 있어요!

5. 자주 쓰는 연산자
   ├─ % : 나머지 (짝수/홀수 판별)
   ├─ == : 같은지 비교
   └─ != : 다른지 비교
```
