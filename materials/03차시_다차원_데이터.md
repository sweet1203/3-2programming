## 🌟 도입 - 우리 반 시간표 떠올리기

여러분의 시간표는 어떻게 생겼나요? 📅

|       | 월   | 화   | 수   | 목   | 금   |
|-------|------|------|------|------|------|
| 1교시 | 국어 | 수학 | 영어 | 정보 | 체육 |
| 2교시 | 수학 | 영어 | 정보 | 국어 | 음악 |
| 3교시 | 영어 | 정보 | 국어 | 수학 | 미술 |

이 시간표는 **가로(요일)** 와 **세로(교시)** 두 방향으로 정보가 있죠?  
이렇게 **2개의 방향**을 가진 데이터를 **2차원 데이터**라고 해요!

> 💡 **차원이란?**  
> - **1차원**: 한 줄 (예: 학생 이름 명단)  
> - **2차원**: 표, 격자 (예: 시간표, 출석부)  
> - **3차원**: 입체 (예: 큐브, 동영상)

## 📚 핵심 개념 1: 1차원 vs 2차원

### 🔹 1차원 리스트 (한 줄)

```python
# 학생 5명의 키 (한 줄로 나열)
heights = [160, 165, 158, 170, 162]
```

이런 식으로 한 줄에 쭉~ 나열된 데이터예요.

### 🔹 2차원 리스트 (표 형태)

```python
# 학생 3명의 정보: [키, 몸무게, 사이즈]
students = [
    [160, 50, 'S'],   # 1번 학생
    [165, 55, 'M'],   # 2번 학생
    [170, 60, 'L']    # 3번 학생
]
```

**리스트 안에 리스트가 들어있죠?** 이게 바로 2차원 리스트예요!

## 📚 핵심 개념 2: 2차원 리스트 접근하기

### 🔹 인덱스 두 개 사용하기

```python
class1 = [[95, 250, "M"], [85, 230, "S"]]
#         └─ 0번 학생 ─┘  └─ 1번 학생 ─┘
#         [0][0] [0][1] [0][2]
```

```python
# 1번 학생(인덱스 0)의 상의 치수(인덱스 0)
print(class1[0][0])   # 95

# 2번 학생(인덱스 1)의 신발 치수(인덱스 1)
print(class1[1][1])   # 230

# 2번 학생(인덱스 1)의 하의 사이즈(인덱스 2)
print(class1[1][2])   # 'S'
```

### 🔹 값 수정하기

```python
class1 = [[95, 250, "M"], [85, 230, "S"]]

# 2번 학생의 상의 치수를 100으로 변경
class1[1][0] = 100
print(class1)
# 출력: [[95, 250, 'M'], [100, 230, 'S']]
```

> 📌 **기억하세요!** `리스트명[행][열]` 순서로 접근해요.

## 📚 핵심 개념 3: 딕셔너리 활용

### 🔹 1차원 딕셔너리

```python
# 한 학생의 정보
mi_song = {'상의': 85, '신발': 230, '하의': 'S'}

print(mi_song['상의'])   # 85
print(mi_song['신발'])   # 230

# 값 수정하기
mi_song['상의'] = 90
print(mi_song)   # {'상의': 90, '신발': 230, '하의': 'S'}
```

### 🔹 2차원 딕셔너리 (딕셔너리 안의 딕셔너리)

```python
class2 = {
    '미송': {'상의': 85, '신발': 230, '하의': 'S'},
    '지우': {'상의': 90, '신발': 240, '하의': 'M'},
    '예린': {'상의': 95, '신발': 250, '하의': 'L'}
}

# 미송이의 모든 정보
print(class2['미송'])
# 출력: {'상의': 85, '신발': 230, '하의': 'S'}

# 미송이의 상의 치수만
print(class2['미송']['상의'])   # 85

# 예린이의 신발 치수
print(class2['예린']['신발'])   # 250
```

> 💡 **딕셔너리의 장점!**  
> 리스트는 `[1][0]` 같은 숫자로 접근하지만, 딕셔너리는 `['미송']['상의']` 처럼 **이름으로** 접근할 수 있어 훨씬 직관적이에요!

## 📌 오늘 배운 내용 한눈에 정리

```
🎁 오늘의 핵심 요약

1. 차원의 의미
   ├─ 1차원: 한 줄 (예: [1,2,3])
   └─ 2차원: 표 형태 (예: [[1,2],[3,4]])

2. 2차원 리스트
   ├─ 만들기: [[행1], [행2], [행3]]
   ├─ 접근: 리스트명[행][열]
   └─ 수정: 리스트명[행][열] = 새값

3. 2차원 딕셔너리
   ├─ 만들기: {'키1': {...}, '키2': {...}}
   ├─ 접근: 딕셔너리[키1][키2]
   └─ 장점: 이름으로 직관적 접근 가능

4. 활용 예시
   시간표, 좌석, 출석부, 점수표, 게임 데이터 등
```
