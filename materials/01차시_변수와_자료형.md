## 📚 핵심 개념 1: 자료형이 뭐예요?

**자료형(Data Type)** 은 "데이터의 종류"를 말해요.  
사람도 학생/선생님/회사원 등 종류가 있듯이, 데이터도 종류가 있어요.

### 🔹 단일형 (Literal) - 값 하나만 담는 자료형

| 자료형 | 영어 이름 | 예시 | 설명 |
|--------|----------|------|------|
| 정수형 | `int` | `100`, `-7`, `0` | 소수점 없는 숫자 |
| 실수형 | `float` | `3.14`, `0.5` | 소수점 있는 숫자 |
| 문자열형 | `str` | `'안녕'`, `"Python"` | 따옴표로 감싼 글자 |
| 부울형 | `bool` | `True`, `False` | 참/거짓 (T, F 대문자!) |

### 🔹 컨테이너형 (Container) - 값 여러 개 담는 자료형

| 자료형 | 영어 이름 | 모양 | 특징 |
|--------|----------|------|------|
| 리스트 | `list` | `[1, 2, 3]` | 대괄호 `[]`, **수정 가능** |
| 딕셔너리 | `dict` | `{'키':'값'}` | 중괄호 `{}`, 키-값 쌍 |

> 💡 **오늘 집중 포인트는 리스트!** 딕셔너리는 아래에서 간단히 맛만 보고, 다음 차시에서 더 자세히 배워요.

---

## 💻 핵심 코드 1: 변수 만들고 자료형 확인하기

```python
# 변수 만들기 (= 는 "오른쪽 값을 왼쪽에 저장하라"는 뜻)
a = 100              # 정수
b = 3.1415           # 실수
c = 'Python'         # 문자열
d = True             # 부울값

# 자료형 확인하기 - type() 함수 사용!
print(type(a))   # 출력: <class 'int'>
print(type(b))   # 출력: <class 'float'>
print(type(c))   # 출력: <class 'str'>
print(type(d))   # 출력: <class 'bool'>
```

### 🔍 코드 풀이
- `print()` : 결과를 화면에 보여주는 함수
- `type()` : 변수의 자료형을 알려주는 함수
- `<class 'int'>` 라고 나오면 "이 변수는 정수 클래스에 속해요"라는 뜻

## 💻 핵심 코드 2: 컨테이너형 자료형

```python
# 리스트 - 무지개 색깔 저장하기
colors = ['빨', '주', '노', '초', '파', '남', '보']
print(colors)         # 전체 출력: ['빨', '주', '노', '초', '파', '남', '보']
print(colors[0])      # 첫 번째 값: 빨  (인덱스는 0부터!)
print(colors[2])      # 세 번째 값: 노  (0→빨, 1→주, 2→노)

# 딕셔너리 - 맛보기만! (이름으로 번호 찾기)
students = {'혜민': '131-9898', '서준': '457-4777'}
print(students['혜민'])   # 혜민이의 전화번호: 131-9898
# 딕셔너리는 3차시에서 자세히 배워요! 지금은 "키로 값 꺼내기"만 기억하세요.
```

### 🔍 중요 포인트
- 리스트의 **순서는 0부터 시작!** (0번, 1번, 2번…)
  - 손가락으로 0, 1, 2… 짚어가며 외우세요!
- 딕셔너리는 **'이름(키)'으로 '값'을 꺼내요** (전화번호부와 같아요)

## 🧩 마무리 퀴즈

### Q1. 다음 중 자료형이 잘못 짝지어진 것은? 🤔
- ① `100` → `int`
- ② `'안녕'` → `str`
- ③ `3.14` → `int`  
- ④ `True` → `bool`

<details><summary>정답 보기</summary>

**정답: ③**  
`3.14`는 소수점이 있으므로 `float`(실수형)입니다.
</details>

### Q2. 다음 코드의 출력 결과는?

```python
colors = ['빨', '주', '노']
print(colors[1])
```
- ① 빨
- ② 주
- ③ 노
- ④ 오류

<details><summary>정답 보기</summary>

**정답: ② 주**  
리스트의 인덱스는 0부터 시작! `colors[0]`이 '빨', `colors[1]`이 '주'.
</details>

### Q3. 빈칸에 알맞은 단어를 쓰세요.

> 딕셔너리는 `{ }`를 사용하며, `_____`와(과) `_____`의 쌍으로 데이터를 저장한다.

<details><summary>정답 보기</summary>

**정답: 키(Key), 값(Value)**
</details>

### Q4. 다음 중 변수의 특징으로 옳은 것은?
- ① 한 번에 여러 값을 저장할 수 있다
- ② 새 값을 저장하면 기존 값이 보존된다
- ③ 한 번에 한 개의 값만 저장하며, 새 값을 저장하면 기존 값은 사라진다
- ④ 변수는 자료형을 미리 선언해야 한다

<details><summary>정답 보기</summary>

**정답: ③**  
파이썬은 자료형을 미리 선언하지 않으며, 변수는 한 번에 하나의 값만 저장합니다.
</details>
