## 🌟 도입 - 일상 속 입출력 찾기

여러분은 매일 카카오톡으로 친구와 대화하죠? 💬

- **입력(Input):** 내가 친구에게 메시지를 **보내는** 것
- **출력(Output):** 친구의 메시지가 화면에 **나타나는** 것

컴퓨터도 똑같아요!
- **입력:** 사용자가 컴퓨터에게 데이터를 주는 것 (`input()`)
- **출력:** 컴퓨터가 사용자에게 결과를 보여주는 것 (`print()`)

> 그런데… 카톡을 끄면 대화가 사라질까요? 안 사라지죠!  
> 어디에 **저장**되어 있기 때문이에요. 컴퓨터에서는 이걸 **"파일"** 이라고 해요. 📁

## 📚 핵심 개념 1: 표준입출력

### 🔹 표준출력 - `print()`
컴퓨터에서 화면으로 **결과를 내보내는** 함수예요.

```python
print('안녕하세요!')           # 안녕하세요!
print('나이:', 17)             # 나이: 17
print('이름:', '민지', '학년:', 2)   # 이름: 민지 학년: 2
```

### 🔹 표준입력 - `input()`
사용자가 키보드로 **값을 입력받는** 함수예요.

```python
name = input('이름을 입력하세요: ')
print('반갑습니다,', name, '님!')
```

> ⚠️ **중요!** `input()` 으로 받은 값은 **항상 문자열(str)** 이에요.  
> 숫자로 쓰려면 `int()`, `float()` 으로 변환해야 해요!

```python
age = input('나이: ')         # 문자열로 받음
age = int(age)                # 정수로 변환
print('내년 나이:', age + 1)
```

## 📚 핵심 개념 2: 파일입출력

### 🤔 왜 파일에 저장할까요?

프로그램을 끄면 변수에 저장한 값은 **모두 사라져요.** 😱  
하지만 파일에 저장하면 **컴퓨터 전원을 꺼도 계속 남아있어요!**

### 🔹 파일 다루는 3단계

```
1️⃣ 열기 (open)    →  2️⃣ 읽기/쓰기 (read/write)  →  3️⃣ 닫기 (close)
```

> 📌 **꼭 기억!** 파일을 열었으면 반드시 닫아야 해요. (책을 보고 덮는 것과 같아요!)

### 🔹 파일 입출력 모드

| 모드 | 영어 | 설명 | 비유 |
|------|------|------|------|
| `'r'` | read | **읽기** 모드 (기본값) | 책 읽기 📖 |
| `'w'` | write | **쓰기** 모드 (기존 내용 삭제!) | 새 노트에 쓰기 ✏️ |
| `'a'` | append | **추가** 모드 (이어서 쓰기) | 일기장에 이어 쓰기 📔 |

> ⚠️ **주의!** `'w'` 모드는 기존 파일 내용을 **모두 지우고** 새로 써요!

---

## 💻 핵심 코드 1: 파일에 쓰기 (`'w'` 모드)

```python
# 1단계: 파일 열기 (없으면 새로 만들어요!)
f = open('연인의_곁.txt', 'w', encoding='utf-8')

# 2단계: 파일에 쓰기
f.write('태양이 바다에 미광을 비추면\n')
f.write('나는 그대를 생각하노라.\n')
f.write('달의 그림자가 샘에 비치면\n')
f.write('나는 그대를 생각하노라.\n')

# 3단계: 파일 닫기 (꼭 닫아야 해요!)
f.close()

print('파일 저장 완료! 📁')
```

### 🔍 코드 풀이
- `open('파일명', '모드')` : 파일을 열어요
- `encoding='utf-8'` : 한글이 깨지지 않게 하는 옵션 (꼭 넣어주세요!)
- `\n` : 줄 바꿈 (Enter와 같아요)
- `f.write()` : 파일에 글자를 써요

---

## 💻 핵심 코드 2: 파일 읽기 (`'r'` 모드)

```python
# 파일 열기
f = open('연인의_곁.txt', 'r', encoding='utf-8')

# 파일 전체 내용 읽기
content = f.read()
print(content)

# 파일 닫기
f.close()
```

### 🔹 한 줄씩 읽기

```python
f = open('연인의_곁.txt', 'r', encoding='utf-8')

# 한 줄씩 반복해서 읽기
for line in f:
    print(line, end='')   # end='' 는 자동 줄바꿈 안 함

f.close()
```

- ① `int`
- ② `float`
- ③ `str`

### Q2. 다음 중 파일 모드의 설명으로 잘못된 것은?
- ① `'r'`: 파일을 읽는 모드
- ② `'w'`: 파일에 새로 쓰는 모드 (기존 내용 삭제)
- ③ `'a'`: 파일 내용을 모두 삭제하는 모드 
- ④ `'w'`: 파일이 없으면 새로 만든다

<details><summary>정답 보기</summary>

**정답: ③**  
`'a'`는 추가(append) 모드로, 기존 내용 **끝에 이어서** 쓰는 모드예요.
</details>

### Q3. 다음 코드를 실행하면 파일에 무엇이 저장될까요?

```python
f = open('test.txt', 'w', encoding='utf-8')
f.write('Hello')
f.write('World')
f.close()
```
- ① `Hello World`
- ② `HelloWorld` 
- ③ `Hello\nWorld`
- ④ 오류 발생

<details><summary>정답 보기</summary>

**정답: ② HelloWorld**  
`\n`이 없으므로 줄바꿈 없이 붙어서 저장돼요.
</details>

## 📌 오늘 배운 내용 한눈에 정리

```
🎁 오늘의 핵심 요약

1. 표준입출력
   ├─ input() : 사용자로부터 값 받기 (결과는 항상 문자열!)
   └─ print() : 화면에 결과 출력

2. 파일 입출력 3단계
   ① open()  : 파일 열기
   ② write()/read() : 쓰기/읽기
   ③ close() : 파일 닫기 (필수!)

3. 파일 모드
   ├─ 'r' : 읽기 (read)
   ├─ 'w' : 쓰기 (write) - 기존 내용 삭제!
   └─ 'a' : 추가 (append) - 이어 쓰기

4. 한글 깨짐 방지: encoding='utf-8'
```
